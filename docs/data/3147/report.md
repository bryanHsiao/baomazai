# 大綜電腦系統（3147）投資研究報告

> 研究基準日：2026 年 7 月 3 日
> 基準股價：236.0 元（昨收 239.0，上櫃 OTC）
> 觸發來源：友人 2026/6/1 分享的「軟體股」明牌之一
> 報告定位：基本面研究，不構成個人化投資建議

---

## 一、投資結論

**結論：中性偏審慎。成長題材強，但估值已高、商業模式毛利低且波動大，追高安全邊際薄。**

**定位更正：3147 是「大綜電腦系統（JETWELL）」，不是軟體股。** 它本質是 **AI 伺服器／資料中心硬體的通路代理＋系統整合（SI）商**（代理 HPE、IBM、Dell EMC、Oracle 等），主機伺服器占營收逾一半、硬體逾八成，真正高毛利的軟體／服務僅約 15~19%。它更接近「**AI 伺服器與半導體資本支出概念的通路／SI 股**」。

2026 年營收爆發驚人（5 月單月年增 812%、前 5 月累計年增 75%），但屬「**案子認列型**」而非穩定經常性收入，單月可暴增暴減、獲利能見度低。以 236 元計，靜態本益比約 28 倍、股價淨值比約 6.4 倍——對一家毛利僅 12~13% 的硬體通路商而言明顯偏貴，市場是用「AI 伺服器高成長」的本夢比在定價。若 AI 拉貨潮趨緩或毛利進一步稀釋，估值下修風險大。

| 項目 | 判斷 |
|---|---|
| 投資評等 | 中性偏審慎／不追高 |
| 風險等級 | 中高（案子型波動＋估值高＋小型股流動性） |
| 核心定位 | AI 伺服器供應鏈通路／SI 股（非軟體股） |
| 每股淨值 | 約 36.7 元 |
| 本益比 | 靜態約 28 倍（2025 EPS 8.43）／前瞻約 17~20 倍 |
| 股價淨值比 | 約 6.4 倍（通路業罕見高檔） |
| 殖利率 | 約 2.8%（已被股價壓縮） |
| 合理價思路 | 保守 126~152／樂觀 180~250（需前瞻 EPS 兌現） |

## 三、業務結構（營收占比與毛利）

2025 年產品結構（法說會口徑）：主機伺服器 **57%**（含 AI 伺服器，2026 Q1 升至 **63%**）、基礎建設軟硬體 24%、顧問服務 10%、解決方案 5%、資安／雲端 4%。

- **軟體／高毛利業務占比小**：顧問＋資安雲端＋解決方案合計僅約 15~19%。公司毛利率被大量「高單價、低毛利的 AI 伺服器」拉低——典型硬體通路／SI 財務特徵，非軟體股高毛利模型。

## 四、題材查證（證據分級）

| 題材 | 級別 | 說明 |
|---|---|---|
| AI 伺服器／GPU 算力／液冷 | **A** | 伺服器占比 57%→63%，營收爆發直接來自 AI GPU 伺服器出貨，有導入直接液冷 |
| 半導體資本支出概念 | **A** | 半導體客戶約占營收 50%，晶圓廠／封測擴廠直接帶動 |
| 記憶體漲價受惠／提前拉貨 | **B** | DDR／SSD 漲價使客戶提前下單、伺服器單價翻倍，屬短期脈衝 |
| 海外擴張（美、馬） | **B** | 2026 才起步，占比尚小 |
| 數位政府／政府標案 | **C** | 查無以政府標案為主的營收，客戶以民間半導體與企業為主 |
| 資安／純軟體 | **C** | 占比僅約 4~8%，非核心、非自研高毛利 |

**釐清**：這不是「數位政府」或「自研軟體」股，而是 AI 伺服器／半導體資本支出的硬體受惠股。

## 八、催化劑與風險

**催化劑**：AI GPU 伺服器與液冷需求、半導體客戶擴廠、記憶體漲價提前拉貨、在手訂單於 2026 Q2/Q3 出貨。52 週高低 352／151.5 元，波動極大。

**主要風險**：
1. **案子型／認列時點波動**——單月營收可暴增暴減（5 月年增 812%），獲利能見度低。
2. **毛利率稀釋**——高單價 AI 伺服器拉低毛利（15.6%→12.6%）。
3. **客戶集中**——半導體約占 50%，綁定台積電供應鏈資本支出循環。
4. **通路商本質**——不掌握核心技術、議價權有限，受品牌原廠與缺料／漲價兩頭夾擊。
5. **小型股流動性**——股本僅 4.37 億，高價股籌碼少、進出衝擊成本高。
6. **估值風險**——PBR 6.4 倍已是通路業罕見高檔，本夢比退潮下修快。

**競爭優勢**：南台灣深耕半導體客群、與晶圓廠／封測綁定深、AI 伺服器＋液冷解決方案卡位、多品牌代理與服務據點的整合維運能力。

## 九、最終判斷

大綜是 AI 伺服器／半導體資本支出的**硬體通路／SI 受惠股**，不是軟體股。2026 營收爆發真實但案子型、波動極大，毛利率下滑，估值（PBR 6.4、靜態 PER 28）已充分反映樂觀情境。適合「AI 伺服器供應鏈題材」的波段思維，不適合「穩定殖利率／低估值」思維（殖利率已降至 3% 以下）。想進場者宜等訂單能見度與毛利率趨勢明朗，或等回檔至前瞻 PER 15 倍以下區間。
