# 規格：佳和實業（1449）完整投資研究

## 研究目的

判斷佳和實業在 2026 年 6 月 8 日時，是否具備可投資價值，並核實市場所稱
「打進特斯拉、無人機、航太」的實質程度。

## 研究原則

- 以公開資訊觀測站、臺灣證券交易所、公司年報及法說資料為第一級來源。
- 新聞及財經網站僅用於補足公司說法、事件脈絡與市場解讀。
- 將「取得認證」、「成為合格供應商」、「取得訂單」、「已產生營收」分開判斷。
- 財務數字需標示期間、單位及是否含新併入子公司。
- 投資結論須包含反方論點、估值情境、催化劑、失效條件及追蹤清單。

## 驗收條件

- [x] 確認公司身分、主要業務、子公司、資本額與管理層。
- [x] 整理至少 2022 至 2025 年營收、EPS、獲利趨勢及 2026 年第一季財務。
- [x] 分析現金流、負債、流動性、增資稀釋與非控制權益。
- [x] 說明 2025 年第四季及 2026 年營收跳升是否來自本業或合併範圍改變。
- [x] 核實特斯拉關係的證據、產品、訂單與營收可見度。
- [x] 核實無人機布局是否已有客戶、訂單、認證或營收。
- [x] 核實航太 AS9100 認證及後續訂單狀態。
- [x] 評估紡織本業、車材、營造與新材料的競爭優勢和主要風險。
- [x] 以最新可得股價建立保守、基準、樂觀三情境估值。
- [x] 提出明確投資評等、合理價區間、買進條件及停看條件。
- [x] 報告中的關鍵事實均附可點擊來源；推論需明確標註為推論。
- [x] 完成事實、算術、日期、來源與前後一致性檢查。
