# 強茂（2481 PANJIT）投資研究報告

> 明牌日：2026 年 7 月 4~5 日（股癌「MOSFET 缺貨」點名、打入 GB200 供應鏈，週末發酵）｜前一交易日（7/3）收盤 207 元（績效基準）
> 研究日：2026 年 7 月 6 日｜盤中約 227.5 元（上市，因題材大漲約 +9.9% 近漲停；報告估值以此為準）
> 報告定位：基本面研究，不構成個人化投資建議

---

## 一、投資結論

**結論：中性偏保守／不宜追高——公司是分立元件老牌大廠、營運回溫是真的，但「打入輝達 GB200 供應鏈」被過度放大，真正驅動力是安世（Nexperia）事件轉單。**

強茂是台灣**分立元件（Discrete）老牌 IDM 大廠**，產品涵蓋二極體（Rectifier/整流二極體全球前列）、蕭特基、TVS、以及 **MOSFET**。這波大漲有兩個題材：①股癌點名 MOSFET 缺貨；②「打入輝達 GB200 供應鏈」。查證後：**MOSFET 只占強茂營收約 15~20%，主力其實是二極體**；「GB200 供應鏈」屬概念/邊緣（分立元件在 AI 伺服器電源用量有限，非核心運算料件），是市場話術。**真正的實質利多是「安世半導體（Nexperia）中荷事件」造成車用/工業分立元件缺口，訂單轉單給強茂等台廠**——這才是 A/B 級的轉單邏輯。現價 227.5 元對應**本益比約 22~25 倍、股價淨值比約 1.8 倍、殖利率約 2.6%**，估值在四檔中相對合理，但今日近漲停、題材（GB200）被誇大，短線追高風險仍高。

| 項目 | 判斷 |
|---|---|
| 投資評等 | 中性偏保守／不宜追高（拉回再評估） |
| 風險等級 | 中（估值相對合理，但題材追高） |
| 本益比 | 約 22~25 倍（近四季 EPS ≈ 9.5） |
| 股價淨值比 | 約 1.8 倍（四檔中最低） |
| 殖利率 | 約 2.6% |
| 合理價思路 | 保守 150~190／中性 190~230 元 |

## 四、題材查證（證據分級）

| 主張 | 級別 | 說明 |
|---|---|---|
| 產業 MOSFET 缺貨漲價 | **A（產業）** | 屬實；但強茂 MOSFET 僅占約 15~20%，主力是二極體 |
| 強茂「打入輝達 GB200 供應鏈」 | **C/D（誇大）** | 分立元件在 AI 伺服器電源用量有限、非核心料件；屬概念性連結，非重大營收驅動 |
| **安世（Nexperia）事件轉單受惠** | **B（實質，最關鍵）** | 中荷 Nexperia 爭端造成車用/工業分立元件缺口，訂單轉台廠——這才是強茂真正利多 |
| 車用/工業分立元件復甦 | **A/B** | 已反映於營收回溫 |

**結論**：市場拿「MOSFET 缺貨＋GB200」當口號，但**強茂的實質受惠來自安世轉單（B 級）與車用復甦（A/B 級）**，GB200 連結是 C/D 級話術。今日大漲是題材連動＋轉單想像。

## 六、風險與競爭優勢

**風險**：①景氣循環（車用/工業去庫存會壓獲利）；②**GB200 題材證據薄弱，若市場醒悟恐回檔**；③安世事件為一次性/事件性利多，未必持續；④紅色供應鏈在二極體低階殺價；⑤今日近漲停追高。

**競爭優勢**：**IDM 垂直整合**（自有晶圓成本控管佳）、二極體全球市占前列、車規認證完整、產品線廣分散風險、切入 SiC、股本大流動性佳、安世事件的直接轉單受惠者。四檔中**基本面最扎實、估值最合理**的一檔。

## 七、最終判斷

強茂是這波四檔裡**體質最穩、估值最合理**的——IDM 老牌大廠、車用比重高、真有安世轉單的實質利多。但市場主打的「MOSFET 缺貨＋GB200 供應鏈」對強茂而言是**次要甚至誇大的題材**（MOSFET 僅 15~20%、GB200 連結薄弱）。若看好，理由應放在「安世轉單＋車用復甦＋估值合理」，而非 GB200。當下近漲停、題材過熱，較佳策略是等拉回到 PER 18~20 倍（約 190~210 元）或轉單/車用復甦於財報驗證後再評估。
